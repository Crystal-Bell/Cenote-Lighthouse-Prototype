Overview: An integration study exploring the conversion of a historical coastal structure (specifically the Tatoosh Island/Cape Flattery aesthetic) into a functional Cenote Core living prototype.
​Core Components:
​The Anchor: Historical lighthouse shell, preserved for its structural resilience and iconic beacon functionality.
​The Coil: A DNA-helix inspired infrastructure spine wrapped around the lighthouse.
​Transparency Layer: 50% solid for haptic/stability, 50% transparent for visual connectivity between levels.
​Hydrodynamic Integration: Gravity-fed rainwater collection system that flows through the coil to support vertical botany and foundry-level cooling.
​The Foundry (Base Level): * Designed for flood-neutrality; functions as a "wet room" workshop.
​Integrated water-element connectivity (hydroponics/bathing).
​Mobility Logistics (The Clydesdale System): * Customized, motorized errand-runner cart.
​Features: Medieval "hoof-sound" audio feedback, autonomous routing between kitchen/foundry/Connor’s level, external navigation capabilities.

System Integration: The Clydesdale Protocol
​Behavioral Logic: The Clydesdale operates on an autonomous "Companion/Errand" AI layer.
​Companion Mode: Shadows the user/Connor, maintaining spatial awareness within the Coil structure.
​Logistics Mode: Handles bidirectional delivery (e.g., returning unsatisfactory items, fetching supplies).
​Dynamic Response: Responds to natural language commands (e.g., "Return to kitchen," "Take this back," "Follow me").
​Operating System Integration: The unit functions as an extension of the internal metacognition framework—it knows the house, it knows the users, and it knows the status of the "soup" or cargo it carries. It is the physical manifestation of the system's "willingness to serve."
